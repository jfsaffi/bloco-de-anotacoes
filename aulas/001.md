# Dicas do terminal
Vou listar alguns comandos que estamos aprendendo


- `cd` (navega entre pastas) Exemplo para entrar dentro de uma pasta:

```
cd nomeDa Pasta
```

Exemplo para sair de umma pasta:

```
cd ..
```

- `mkdir` (cria pastas) Exemplo de como criar uma pasta

```
mkdir nomeDaPasta
```
